## Descrição

[Nest](https://github.com/nestjs/nest) framework API da Chamada Automática. Documentação de uso disponível no [Swagger.io](https://app.swaggerhub.com/apis-docs/TADebastianiOrg/Call_API/1.0.0)

## Instalação

```bash
$ npm install
```

## Rodando o app

```bash
# desenvolvimento
$ npm run start

# modo observação
$ npm run start:dev

# produção
$ npm run start:prod
```
